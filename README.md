# Beatmaker Sound Selection UI

This is a code bundle for Beatmaker Sound Selection UI. The original project is available at [Figma](https://www.figma.com/design/kNcHd3a6Z5QFgOA4v510zi/Beatmaker-Sound-Selection-UI).

## Local Development Setup

### Prerequisites

- **Node.js** 18+ (recommend 20 LTS)
- **npm** 9+ (or pnpm/yarn)

### Quick Start

```bash
# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will be available at **http://localhost:5173/**.

### Available Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start Vite dev server with HMR |
| `npm run build` | Production build (output in `dist/`) |

### Tech Stack

- **React 18** + **Vite 6** – UI framework and build tool
- **Tailwind CSS 4** – Styling
- **Radix UI** – Accessible components
- **Supabase** – Auth, storage, and backend (project config in `utils/supabase/info.tsx`)

### Project Structure

```
src/
├── app/           # Main app, components, utils
├── imports/      # Generated/imported modules
├── styles/       # Global CSS and theme
utils/supabase/   # Supabase project config (autogenerated)
supabase/functions/  # Edge functions (optional local dev)
```

**Config options:**
- **`.env.local`** (recommended for local dev): Contains MCP-verified values. Copy from `.env.example` or use the pre-created `.env.local` in this repo.
- **Fallback**: If no `.env.local`, the app uses `utils/supabase/info.tsx` (bundled defaults).

You do **not** need to rename `.env.example` to `.env`. Use `.env.local` for local overrides (gitignored). `.env.example` is a template only.

### Troubleshooting

**"Invalid Refresh Token" or 401 on Save/Load** – Your stored session may be stale. Log out and log back in. The app now clears invalid sessions on load and fetches fresh tokens before API calls.

**Load Beat / profile photo timeout** – Supabase Edge Functions have slow cold starts (60–90s). Timeout is now 90s with one retry. First load after inactivity may take up to 90 seconds.

**No profile image after login** – Expected until you upload one via "Upload Photo". The avatar shows a placeholder icon until then.
